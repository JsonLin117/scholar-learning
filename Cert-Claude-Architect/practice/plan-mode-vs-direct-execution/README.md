# Plan Mode vs Direct Execution 練習

## 場景
ODM 供應鏈 DE 團隊使用 Claude Code，需要為不同任務選擇正確的執行模式。

## 練習內容
1. **Permission Mode 系統模擬**：完整的 5 種 mode 切換邏輯
2. **Task Router 決策引擎**：根據任務特徵（檔案數、複雜度、codebase 熟悉度）自動推薦 Plan/Direct
3. **Explore Subagent 模擬**：context 隔離、thoroughness 三級、verbose 輸出壓縮
4. **Ultraplan 工作流模擬**：CLI → Cloud → Review → Teleport Back
5. **Iterative Refinement 模式選擇器**：根據情境推薦 I/O examples / TDD / Interview / Batch
6. **CCA Mock Exam**：5 題 D3 Task 3.4 模擬考
