# Task Decomposition & Session Management Simulator

## 場景
模擬 CCA 考試中的三個核心概念：
1. **固定循序 vs 動態適應分解** — 根據任務特徵選擇正確策略
2. **PostToolUse Hook 跨 MCP 正規化** — 統一多源數據格式
3. **Session 策略選擇** — resume vs fork vs fresh start

## 練習內容
- Task Decomposition Strategy Selector: 15 個場景自動判斷
- Cross-MCP Normalizer: 模擬 3 個 MCP Server 返回異質格式
- Session Strategy Decision Matrix: 8 個場景判斷
- Coordinator Decomposition Validator: 偵測分解過窄
- 5 題 CCA 模擬考
